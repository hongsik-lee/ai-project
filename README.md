# AI Project

3개의 프로젝트를 하나의 저장소로 관리하는 모노레포입니다.

## 폴더 구조

- `ai-board-project` : 기존 게시판 프로젝트 (Vite + Vanilla JS + Sass)
- `ai-react-project` : React 프로젝트 (현재 작업 중)
- `ai-view-project` : 추후 개발 예정 프로젝트

## 프로젝트별 실행

### 1) ai-board-project

```bash
cd ai-board-project
npm install
npm run dev
npm run api
```

### 2) ai-react-project

```bash
cd ai-react-project
npm install
npm start
```

### 3) ai-view-project

추후 생성 예정
